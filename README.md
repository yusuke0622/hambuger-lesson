# $(this).toggleClass('active');
  .toggleClass()は、指定されているclass（ここではactive）をトグル処理するメソッド  
  トグル処理では、該当のclass属性がある場合にはclassが削除され、ない場合にはclassが追加される（🟰アイコンが切り替わる）
